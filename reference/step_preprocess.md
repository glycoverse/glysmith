# Step: Preprocessing

Preprocess the experiment using
[`glyclean::auto_clean()`](https://glycoverse.github.io/glyclean/reference/auto_clean.html).

## Usage

``` r
step_preprocess()
```

## Value

A `glysmith_step` object.

## Examples

``` r
step_preprocess()
#> <step "preprocess"> Preprocessing
```
