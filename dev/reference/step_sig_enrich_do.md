# Step: DO Enrichment Analysis on Differentially Expressed Variables

Perform Disease Ontology enrichment analysis on differentially expressed
variables using
[`glyfun::enrich_ora_do()`](https://glycoverse.github.io/glyfun/reference/enrich_ora_do.html).

This step requires `dea_res` (differential analysis result from DEA).
Run one of
[`step_dea_limma()`](https://glycoverse.github.io/glysmith/dev/reference/step_dea_limma.md),
[`step_dea_ttest()`](https://glycoverse.github.io/glysmith/dev/reference/step_dea_ttest.md),
or
[`step_dea_wilcox()`](https://glycoverse.github.io/glysmith/dev/reference/step_dea_wilcox.md)
before this step. Only execute for glycoproteomics experiments with
exactly 2 groups. If used for glycomics experiments, the step will be
skipped. Use all genes in OrgDb as the background.

## Usage

``` r
step_sig_enrich_do(
  universe = "all",
  plot_type = "dotplot",
  plot_width = 7,
  plot_height = 7,
  ...
)
```

## Arguments

- universe:

  The universe (background) to use for enrichment analysis. One of "all"
  (all genes in OrgDb), "detected" (detected variables in `exp`).

- plot_type:

  Plot type for enrichment results ("dotplot", "barplot", etc.).

- plot_width:

  Width of the plot in inches. Default is 7.

- plot_height:

  Height of the plot in inches. Default is 7.

- ...:

  Additional arguments passed to
  [`glyfun::enrich_ora_do()`](https://glycoverse.github.io/glyfun/reference/enrich_ora_do.html).

## Value

A `glysmith_step` object.

## Details

Data required:

- `exp`: The experiment to perform Disease Ontology enrichment analysis
  for

- `dea_res`: The result from DEA, generated by `step_dea_xxx()`.

Tables generated:

- `do_enrich`: A table containing the Disease Ontology enrichment
  results.

## See also

[`glyfun::enrich_ora_do()`](https://glycoverse.github.io/glyfun/reference/enrich_ora_do.html)

## Examples

``` r
step_sig_enrich_do()
#> <step "step_sig_enrich_do()"> DO enrichment analysis
step_sig_enrich_do(plot_type = "barplot")
#> <step "step_sig_enrich_do(plot_type = \"barplot\")"> DO enrichment analysis
```
